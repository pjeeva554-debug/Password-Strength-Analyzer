# 🔐 Password Strength Analyzer (Java Project)



## 📌 Overview

This is a simple Java console-based project that analyzes the strength of a user-entered password.  

It checks different security rules and classifies the password as Weak, Medium, or Strong.



---



## 🚀 Features

- Checks password length (minimum 8 characters)

- Checks presence of uppercase letters (A–Z)

- Checks presence of lowercase letters (a–z)

- Checks numbers (0–9)

- Checks special characters (!@#$%^&*())

- Calculates a password strength score

- Displays final password strength result



---



## 🧠 Logic Used

Each condition adds **1 point** to the score:



- Length ≥ 8 → +1  

- Uppercase letter → +1  

- Lowercase letter → +1  

- Number → +1  

- Special character → +1  



---



## 📊 Password Strength Rules

- Score 0–2 → Weak Password ❌  

- Score 3–4 → Medium Password ⚠️  

- Score 5 → Strong Password 🔐  



---



## 💻 Technologies Used

- Java Programming Language  

- Scanner Class (User Input)  

- String Methods  

- Regular Expressions (Regex)  

- Conditional Statements (if-else)



---



## ▶️ How to Run

1. Open the project in any Java IDE (VS Code / Eclipse / IntelliJ)

2. Compile the program

3. Run the file

4. Enter a password when prompted

5. View the strength result
